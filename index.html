Prework Study Guide

HTML

  • HTML is what makes up the content structure of the web page.
  • HTML is broken down from the head, header, to body to footer elements.
  • The ul tag creates unordered lists, while the ol tag creates ordered lists.
  • You can organize your HTML code by using the "section" tag.

GIT

  • You can use the command cd to change directories.
  • Command ls is used to view the contents of the directory you are on.
  • The git status command checks which branch you are in.
  • Git checkout -b feature/branch-name: is a command to create a new branch.

CSS

  • CSS stands for "cascading style sheets," and helps style the application.
  • You can target html elements in CSS or class attributes.
  • A primary principal when programming is the "DRY" principal meaning dont repeat yourself.
  • The "bod model" consist of content, padding, border then margin.
